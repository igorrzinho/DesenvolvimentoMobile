 Instalar o expo na maquina: 
``` bash
npm install -g expo-cli
```
 Criar um projeto: 
``` bash
npx create-expo-app MeuApp --template
# ou crie na pasta local 
npx create-expo-app . -t
```
 Rode com:
``` bash
cd app_exemplo
npx expo install react-dom react-native-web
npm start
```
